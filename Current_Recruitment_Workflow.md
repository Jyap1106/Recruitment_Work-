# Recruitment Operating System (Current Workflow)

# Recruitment Technology Stack

| System | Purpose |
|---------|---------|
| Moka | Applicant Tracking System (ATS) |
| LinkedIn Recruiter | Candidate sourcing and outreach |
| JobStreet | Job posting and applications |
| Company Career Page | Direct applications |
| Internal Referral Portal | Employee referrals |
| WhatsApp | Primary communication channel (Malaysia & Singapore) |
| WhatsApp Call | Initial recruiter screening call / interview scheduling |
| Lark | Online interviews for overseas candidates |
| Calendar | Interview scheduling |
| Email | Candidate confirmation and formal communication |
| Recruitment Masterfile | Recruitment tracking |
| Personal Position Tracker | Recruiter workload tracking |
| Candidate Onboarding Tracker | New hire onboarding tracking |

## Overview

This document outlines the current end-to-end recruitment workflow as performed manually. It serves as the baseline process before introducing AI assistance and workflow automation.

---

# Module 1 – Job Requisition & Job Advertisement

## Trigger
Approved Job Requisition received through Moka.

### Process

1. Receive approved Job Requisition.
2. Review hiring requirements.
3. Publish the position on:
   - LinkedIn
   - JobStreet
   - Company Career Page
   - Internal Referral Portal
4. Every Thursday:
   - Review all active vacancies.
   - Prepare a list of positions for company-wide communications.
   - Retrieve the referral link for each position.
   - Submit the list to the Communications team.

---

# Module 2 – Candidate Attraction & Initial Screening

## Input

Applications received from:
- LinkedIn Recruiter
- LinkedIn Job Posting
- JobStreet
- Company Career Page
- Internal Referral Portal

### Process

1. Review candidate profile.
2. Assess suitability.
3. Shortlist suitable candidates.

### Candidate Outreach

Primary communication channel:

Malaysia / Singapore
- WhatsApp message
- Follow-up WhatsApp call if required

International candidates
- LinkedIn message (if applicable)
- Lark interview invitation

### Interview Scheduling

1. Agree on interview timing.
2. Schedule Recruiter Interview.
3. Block 30-minute calendar slot.
4. Send:
   - WhatsApp confirmation
   - Lark interview link (international candidates only)

---

# Module 3 – Recruiter Interview & Assessment

### Process

1. Conduct Recruiter Interview.
2. Take shorthand interview notes.
3. Rewrite interview notes into structured format.
4. Generate:
   - Full Candidate Summary
   - Condensed Candidate Summary
5. Review generated summary.
6. Update ATS records:
   - Interview notes
   - Recruiter assessment
   - Suitability
   - Recommendation
7. Decide:
   - Reject
   - Proceed to Hiring Manager

---

# Module 4 – Hiring Manager Assessment

### Process

1. Candidate enters Hiring Manager shortlist.
2. Contact candidate to arrange interview timing.

Communication Channel

Malaysia / Singapore
- WhatsApp

International
- Lark

3. Schedule interview in Moka.
4. Moka generates interview link.
5. Candidate attends interview.
6. Hiring Manager submits:
   - Interview feedback
   - Interview score
7. Decide:
   - Reject
   - Proceed to Offer Stage

---

# Module 5 – Offer Management

### Process

1. Conduct Pre-Offer Call.
2. Confirm:
   - Candidate interest
   - Salary expectation
   - Concerns
3. Discuss salary proposal internally with:
   - Hiring Manager
   - HRBP
4. Submit Offer Proposal in Moka.
5. Await approval.
6. Contact candidate.
7. Present verbal offer.
8. Candidate decides:
   - Accept
   - Reject

---

# Module 6 – Pre-Onboarding

### Trigger
Candidate accepts offer.

### Process

1. Collect required documents.
2. Upload documents into system.
3. Update:
   - Recruitment Masterfile
   - Personal Position Tracker
   - Candidate Onboarding Tracker
4. Notify internal stakeholders.
5. Send calendar invitations:
   - Hiring Manager
   - IT Team
6. Send confirmation email to candidate.
